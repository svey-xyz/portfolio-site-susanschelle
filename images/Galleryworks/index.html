Information for New Web Accounts

Welcome to the start page for new web accounts on www.ryerson.ca. This page provides you with general information about your account and help to get you started in developing your web site. Please read through for Ryerson specific setup options.

Accessing your Account:

  • The current URL or address for your site is http://www.ryerson.ca/~accountname For department or research accounts you can request an alias to remove the tilde(~) from the address and extend the accountname to be more descriptive. Please contact the CCS Help desk at extension 6806 for this. The alias option is NOT available for personal accounts.
  • Connect to the www.ryerson.ca server to place your Web files online or to check on the status of your account.
    • To upload your Web files, use Ftp(File Transfer Protocol) or a Web page development package such as Dreamweaver.
    • To change your account password, use SSH software to connect to www.ryerson.ca and login. Select option 1 to change your password.
  • Each account is allocated a storage space quota. When you reach the quota, your account locks which prevents any further updates. If you are uploading large files, check your space usage. Use SSH to connect to www.ryerson.ca and your quota is displayed on the main screen.

Web Account File Structure:

When creating your web site there are a few points to keep in mind. Your web account has a directory called public_html. All files to be used on your web pages should be placed in this directory. The first file you need to create is the index.html file which is the file you are currently viewing on your new account. The index.html is the default page displayed in any directory. When creating a new directory on your account make sure to create an index.html. If you remove the index.html all your files will be listed when the directory is accessed by a web browser.

Note: To access the information found on this page connect to the CCS Web Site help page.
